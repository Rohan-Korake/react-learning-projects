## Getting Started

Clone the repository:

```bash
git clone https://github.com/Rohan-Korake/react-learning-projects.git
```

Navigate to a project:

```bash
cd project-folder
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
